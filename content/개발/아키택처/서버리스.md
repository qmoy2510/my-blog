## 서버리스란?
- 기존에 존재하던 컴퓨터를 제공해주는 cloud 방식은 컴퓨터만 제공해줄뿐 설정은 전부 개발자가 해야했음 이 점을 보완해서 등장한 서비스가 서버리스임
- 개발자는 개발에 집중할 수 있도록 서비스 제공자 측에서 os 설정, 컴퓨터 관리, 트래픽 관리 등 서버 영역은 전부 케어해줌
- 서버 리스가 주로 쓰이는 서비스는 aws lambda 등의 function 서버에서도 많이쓰임
